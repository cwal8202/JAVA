폰트설정
window - preference - 검색창에 font검색 - colors and font 
basic 확장시키면 text font있다 - 더블클릭 후 크기와 글꼴 바꾸기

encoding 설정

encoding검색 - workspace 클릭 - text file encoding 에서 default 되어 있는걸
other 클릭 후 utf-8로 바꾸기! - 어플라이 앤 클로즈 하면 설정 되었다!

파일 하나는 프로젝트라고 한다.

프로젝트는 보통 workspace 위치에 저장된다.
혹시라도 다른곳에 저장하고 싶으면 use default location 클릭하기!

파일 만들어지면 파일아이콘에 J가 붙어있을 것이다
파일 확장하면 scr가 있을것인데 src가 소스코드 폴더이며 소스코드가 다 저장된다.

src 우클릭 new마우스 갖다대면 class나온다
name 에 이름적기

public static void main 클릭하면
자동으로 메인메소드 작성해준다.
