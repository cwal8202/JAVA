부모의 생성자를 상속받은 자식 생성자는 부모생성자의 형태를 상속받는다

부모생성자는 super();라고 표현한다.

자식 생성자(부모필드,부모필드,자식필드) {
	super(부모의 생성자형태);
	this.자식필드 = 자식필드;
}

자식 기본생성자 () {
	super();
	this.자식필드 = null;
}

자식의 생성자는 부모생성자(super)를 무조건 가져와야하는데
부모생성자가 기본생성자면 기본생성자를 쓰고
부모생
